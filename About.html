Here I'll tell you how this page works. I've written the front-end using the framework called bootstrap. It allowed me to finish quickly the elements. On the first page, there is a button that activate the listener behind the scenes.

The first action that this listener does is to get the string written in the form. For that, I used DOM manipulation. Then I set this string to a variable called form text which I pass through a parameter to a function called morse code. This function runs character by character of this string to check each element, so, if the first element of formText is an 'a' the function will write '.-' in a new string. Working this way using a loop I can read any string and translate it to morse code.
